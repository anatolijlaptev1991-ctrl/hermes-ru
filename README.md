# 🇷🇺 hermes-ru

Русская локализация **Hermes Agent Desktop** для Windows.

Полный перевод интерфейса (меню, настройки, чат, уведомления, инструменты).
Локализация **не слетает при обновлениях Hermes** благодаря self-healing launcher.

## Возможности

- ✅ Полный перевод интерфейса
- ✅ Не слетает при обновлениях Hermes (self-healing launcher)
- ✅ Восстановление в один клик (`hermes-ru uninstall`)
- ✅ Исходный код включён в репозиторий

## Установка

### Требования
- Windows 10 / 11
- [Node.js](https://nodejs.org) 18+
- Установленный Hermes Desktop

### Способ 1 — через npm (рекомендуется)

```bash
npm install -g hermes-ru
hermes-ru install
```

### Способ 2 — без npm (локально)

1. Скачайте `hermes-ru-v0.17.0.zip` на странице [Releases](../../releases)
2. Распакуйте
3. Откройте терминал в папке и выполните:

```bash
npm install
node bin/cli.js install
```

## Запуск

Откройте меню Пуск → **«Hermes (Русский)»**

> Ярлык создаётся автоматически при `hermes-ru install`.
> При обычном запуске Hermes (без ярлыка) перевод может слететь после обновления —
> используйте ярлык «Hermes (Русский)» для авто-восстановления.

## Команды

| Команда | Описание |
|---------|----------|
| `hermes-ru install` | Установить русскую локализацию |
| `hermes-ru status` | Проверить статус |
| `hermes-ru repair` | Перепатчить (после обновления Hermes) |
| `hermes-ru uninstall` | Вернуть английский интерфейс |

## Как это работает

1. `hermes-ru install` находит `Hermes.exe`, распаковывает `app.asar`,
   заменяет `dist/` на версию с русским переводом, запаковывает обратно.
2. Перевод копируется в `~/.hermes/russian-loc/` — переживает обновления.
3. В меню Пуск создаётся ярлык **«Hermes (Русский)»**.
4. При запуске через ярлык launcher проверяет целостность перевода.
   Если Hermes обновился и перезаписал asar — launcher автоматически
   перепатчивает его и запускает Hermes.

## Структура репозитория

```
hermes-ru/
├── dist/              # Предсобранный перевод (готовый к установке)
├── src/i18n/          # Исходный код локализации (ru.ts + патчи)
├── bin/               # CLI и patcher
├── launcher/          # Self-healing launcher
└── compat.json        # Карта совместимости версий
```
